Shipping Information

How to Package It

  1. Take out batteries (no need to ship them)
  2. Lock pendulum or remove and wrap separately
  3. Wrap clock in bubble wrap
  4. Fill in open space with peanuts, bubblewrap, newspaper etc. (There should be no movement in box when shaken)
  5. Put clock in box with face up towards the shipping label
  6. Include packing slip (found below) inside box

    7. **Double box if possible

  7. Put first box inside of larger box
  8. Fill space between boxes with peanuts, bubblewrap, newspaper etc. (Smaller box should not be touching larger box)

How to Ship It

We recommend using FedEx to ship

  • We will send it back using FedEx
  • They can box it for you
  • They can include a return label

ALWAYS insure your package

  • We recommend insuring for at least $500
  • We can give you a recommendation for your specific clock

What to Include

Please include a completed packing slip in the shipment.
